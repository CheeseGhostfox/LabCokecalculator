# LabCokecalculator
A automated calculator for some one who wants to try out "LabCoatz" 's "Labcoke" making.
this app is for MANDARIN users only,It's just for fun,I can't do more localization for other languages users TwT.
一个自动化的配方计算器,为那些想要本地复现"LabCoatz"频道的"Labcoke"可乐配方(https://www.youtube.com/watch?v=TDkH3EbWTYc)的化学大手子设计.
它可以帮助您快速计算与生成不同容积,不同版本的配方与制作方法---在一个巨大的玻璃烧杯里搅拌糖浆与苏打可比做数学有趣多了!
同时包含三种不同的变体---以等甜度三氯蔗糖替换50%白砂糖的减糖版,以等酸度维生素C与柠檬酸替代磷酸的无磷酸版,以及不含咖啡因的"和谐版"的自动计算.
